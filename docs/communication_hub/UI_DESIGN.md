# Communication Hub UI Design

## Visual Paradigm Decision

After evaluating multiple approaches, we've chosen a **FAB (Floating Action Button) + Expandable Panel** pattern.

### Why FAB?

| Approach | Pros | Cons | Decision |
|----------|------|------|----------|
| Persistent Sidebar | Always accessible | Takes screen real estate | Rejected |
| Card-based | Consistent with app | Feels disconnected, gets lost | Rejected |
| Contextual Strip | Visible but minimal | Novel pattern, limited space | Considered |
| **FAB + Panel** | Eye-catching, non-intrusive, mobile-friendly | Extra click to access | **Selected** |

### Why "Hub" Not "Chat"

A chat bubble icon (ğŸ’¬) limits user perception. The hub contains much more:
- Messages
- Activity
- Notifications
- Presence
- Quick actions
